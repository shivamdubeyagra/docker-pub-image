# Hello Docker – Run This Project

## **1. Pull the Docker image**

```
docker pull shivamdubeyagra/hello-docker:latest
```

## **2. Run the container**

```
docker run -p 3000:3000 shivamdubeyagra/hello-docker:latest
```

## **3. Open in browser**

```
http://localhost:3000
```
You will see:
```
Hello World!
```

That's it. ✔
